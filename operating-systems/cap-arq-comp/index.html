Tipos de procesadores

Dependendiendo de la cantidad de ciclos por instrucción (CPI), los procesadores se pueden clasificar en:

  • Sub-escalares
  • Escalares
  • Super-escalares
Se asume que el ciclo de ejecución de una instrucción está dividida en cinco etapas:
  • Instruction Fetch (IF)
  • Instruction Decode (ID)
  • Execution (EX)
  • Memory access (MEM)
  • Write Back (WB)

Deficiones

Physical Procesor (PP)

Es la implementación en HW de una unidad de procesamiento. Actualmente, un core es un PP.

Simultaneous Multithreading (SMT)

Tecnología que permite duplicar recursos de un PP, como el conjunto de registros, PC, mecanismo de interrupciones, etc. Permite la ejecución de instrucciones de múltiples software threads en un sólo ciclo en un PP. El procesador debe ser super-escalar para soportar esta tecnología.

Esto significa que un PP puede tener múltiples contextos de ejecución.

Logical Procesor

Unidad de procesamiento que mantiene un sólo contexto de ejecución (hardware thread)

Un LP puede ser un PP (core). Cuando hay SMT, hay tantos LP como capacidad de ejecución de threads existan.
